Arrays

Método join

Genera un string que contiene los substrings mantenidos por cada elemento del array unidos por el ó los caracteres indicados en el parámetro separador.
Si no se utiliza el parámetro, este método utiliza la coma (,) como separador.
Concatena los elementos de un array en una sola cadena separada por un carácter opcional

Nombre_array.join([separador])
