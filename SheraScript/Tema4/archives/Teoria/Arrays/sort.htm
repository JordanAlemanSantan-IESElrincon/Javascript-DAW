Arrays

Método sort

Nombre_array.sort([función])

Ordena los elementos del array.
Si no se utiliza el parámetro, este método ordena los elementos del array de menor a mayor comparando cada elemento como si se tratase de una string.
Si se pasa como parámetro una función de ordenación, este método ordena los elementos del array de acuerdo con el criterio de ordenación de la función.
