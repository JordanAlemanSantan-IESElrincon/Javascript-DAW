Arrays

Método push y método pop

Método pop

Nombre_array.pop()

Elimina y devuelve el último elemento del array. Este método modifica la longitud del array (atributo length).

Método push

Nombre_array.push(elem1, ... elemN)

Añade al final del array los elementos que se pasan como parámetro y devuelve la nueva longitud de éste.
