Diferentes loops

Ejercicio 3

Realizar un programa que contenga las diferencias entre los loops:

  • for
  • forEach
  • for in
  • for of

En el mismo código explicar las diferencias, pros y contras, de cada uno de ellos.

Loop for

El bucle for cumple la siguiente sintaxis.
for (statement 1; statement 2; statement 3) {}, todos valores opcionales donde:
statement 1: Se ejecuta antes de iniciarse el bucle. Normalmente aquí se inicia una variable como un contador (i = 0).
statement 2: Define la condición de salida del bucle (i <= 5).
Si no se indica ninguna condición, habrá que poner como mínimo un ';' en este statement y dentro del bucle un 'break' para indicarle cuando saldrá del bucle, o si no sería un bucle infinito.
statement 3: Se ejecuta después de cada vuelta del bucle. Normalmente se suele Incrementar o Decrementar la variable que hemos inicializado en el primer statement (i++).

Tiene varios usos este bucle for, tanto para cálculos matemáticos como para realizar recorridos como en un array.

Recorriendo un array

let fruits = ["apple", "orange", "cherry", "avocado", "banana", "cherries"];
let log = "";

for (i=0; i < fruits.length; i++) {
    log += "<b>" + i + ": </b>" + fruits[i] + "<br>";
}

Cálculo numérico

let log = "";

for (i = 0; i <= 5; i++) {
    log += "<b>Número: </b>" + (i * 5) + "<br>"
}

Loop forEach

El bucle forEach es más bien una función que realiza el recorrido de un array.
La sintaxis es forEach(function(currentValue, index, arr)) donde:
currentValue: Corresponde al valor en el recorrido del array.
index: Corresponde a la posición en el array.
arr: Corresponde al array que se está recorriendo.

Recorriendo un array

let fruits = ["apple", "orange", "cherry", "avocado", "banana", "cherries"];
let log = "";

fruits.forEach((item, index) =>
    log += "<b>" + index + ": </b>" + item + "<br>";



Actualizando los valores de un array

let fruits = [4, 8, 15, 16, 23, 42];
let log = "";

fruits.forEach((item, index, arr) => {
    log += "<b>" + index + ": </b>" + item + " * 10 => "
    arr[index] = item * 10;
    log += item + "<br>"
});

Loop for in

El bucle for in se realiza a través de las propiedades de un objeto.
Podremos ver el nombre de las propiedades de cada objeto o las posiciones de nuestros arrays, o podremos ver las propiedades en sí de cada objeto como el contenido de nuestros arrays.

Propiedades de un objeto

let person = {fname:"John", lname:"Doe", age:25};
let text = "";

for (let index in person) {
    text += "<b>" + index + ":</b> " + person[index] + "<br>";
}

Posiciones y contenido de un array

let cars = ['BMW', 'Volvo', 'Mini'];
let text = "";

for (let index in cars) {
    text += "<b>" + index + ":</b> " + cars[index] + "<br>";
}

Loop for of

El bucle for of recorre los valores de un array o de una String. En el caso de una String, recorrerá caracter por caracter.

En ambos casos accederemos directamente a su valor a través de la variable que crearemos dentro del bucle for of que llamaremos 'valor' y mostraremos los resultados con esta variable al mostrarla sin más y al mostrarla como posición del array o del String.

Recorriendo un array

let cars = ['BMW', 'Volvo', 'Mini'];
let text = "";

for (let valor of cars) {
    text += "<b>Posición: </b>" + cars[valor] +
" ----------- <b>Valor: </b>" + valor + "<br>";
}
